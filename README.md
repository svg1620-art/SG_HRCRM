# SG HRCRM

CRM для обработки входящих откликов hh.ru: настраиваемый скоринг по вакансии, короткий диалог и автоматическое движение кандидата по воронке.

## Локальный запуск

```bash
pnpm install
pnpm dev
```

Скопируйте `.env.example` в `.env` и добавьте ключи hh.ru, OpenAI и PostgreSQL. Production-домен приложения: https://hrcrm.serviceguru.pro. Пока ключей нет, интерфейс работает с демонстрационными данными.

## Интеграция hh.ru

1. Зарегистрируйте приложение на https://dev.hh.ru.
2. Укажите callback URL `https://hrcrm.serviceguru.pro/api/hh/callback`.
3. Авторизуйте менеджера работодателя по OAuth.
4. Синхронизируйте коллекции входящих откликов по каждой активной вакансии.

Сообщения hh.ru поддерживаются через API откликов, но hh.ru помечает эти методы как устаревающие. Интеграционный слой нужно держать изолированным и предусмотреть переход на внешний канал.

## Railway

Проект содержит `railway.json`. Добавьте сервис PostgreSQL, переменные из `.env.example` и подключите GitHub-репозиторий. Railway выполнит сборку и запустит приложение автоматически.
